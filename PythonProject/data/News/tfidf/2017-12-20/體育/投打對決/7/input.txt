小熊 補強 先發   總裁 面會 達比修 小熊 為了 補強 先發 輪值 ， 認真 考慮 簽下 達比修有 。 芝加哥 當地 媒體 記者 李文 ( Bruce   Levine ) 表示 ： 「 小熊隊 的 高層 ， 包括 總裁 艾普斯坦 ( Theo   Epstein ) 、 總經理 霍 耶 ( Jed   Hoyer ) ， 已 到 德州 與 達比修有 會面 。 」 
 專欄 作家 海曼 ( Jon   Heyman ) 表示 ： 「 小熊 想要 補強 先發 輪值 ， 其 中 有 幾 個 目標 ， 包括 達比修有 、 艾 瑞亞 塔 ( Jake   Arrieta ) 、 阿契 ( Chris   Archer ) 、 林恩 ( Lance   Lynn ) 都 是 選擇 ， 但 小熊 已 先 跟 達比修有 見面 。 」 
 小熊 在 休季 目標 補強 先發 ， 以 3 年 3800 萬 美元 ( 約 11.25 億 台幣 ) 簽下 的 右投 查特 伍德 ( Tyler   Chatwood ) 後 ， 先發 輪值 還 缺 1 人 。 小熊 是 去年 冠軍 ， 達比修 也 想 加入 有 競爭力 的 球隊 ， 雙方 似乎 有 機會 合作 。 
 ( 記者 王翊 &# 20120 ; )
